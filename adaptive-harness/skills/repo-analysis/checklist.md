# Repo Analysis Checklist

- [ ] Directory tree collected to configured depth
- [ ] Excluded directories skipped (node_modules, .git, dist, etc.)
- [ ] Languages detected from file extensions
- [ ] Package manager identified from lockfiles
- [ ] Frameworks detected from dependency manifests
- [ ] Commands extracted from package scripts or Makefile
- [ ] CI/CD files identified
- [ ] Environment files catalogued
- [ ] Risk areas identified (auth, payments, security, env, deployment, database)
- [ ] Adapters detected with confidence scores
- [ ] Profile saved to memory store
- [ ] No files modified during analysis
