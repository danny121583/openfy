# @orbitos/core-types

TypeScript interfaces and type contracts for OrbitOS Core Runtime objects, processes, and capabilities.

## Installation

This is an internal package used by the OrbitOS core runtime packages:

```json
"dependencies": {
  "@orbitos/core-types": "workspace:*"
}
```

## Basic Types Excluded

Includes definitions for:
*   `OrbitBaseObject`
*   `OrbitObjectType`
*   `OrbitDeviceType`
*   `ApprovalLevel`
*   `CapabilityScope`
*   `OrbitRelationship`
*   `OrbitActivityNode`
*   `CapabilityRule`
